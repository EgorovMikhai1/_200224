JSON (JavaScript Object Notation) — это текстовый формат обмена данными,
основанный на JavaScript. Как и YAML, он часто используется для
передачи данных в веб-приложениях и конфигурационных файлах
благодаря его удобочитаемости и простоте использования.
JSON поддерживает базовые типы данных: строки, числа, объекты
(словари или хэш-таблицы), массивы, булевы значения и null.

Вот пример JSON-документа:

json
Copy code
{
  "name": "John",
  "age": 30,
  "city": "New York",
  "skills": ["Python", "Machine Learning", "AI"]
}
JSON используется во многих современных веб-API в качестве
формата обмена данными и считается стандартом для обмена данными
между клиентом и сервером.

JSON обычно парсится и преобразуется в соответствующие структуры
данных в различных языках программирования, таких как JavaScript,
Python, Java и других, что позволяет разработчикам легко работать с данными.
